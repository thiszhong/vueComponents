# vue-components

## Project setup
```
npm install
```

### Compiles and hot-reloads for development 预览组件
```
npm run serve
```

### Components:
 1. ProductItem（商品的一个item），内部以em为单位，可以适应环境的适配，使用时设置外层 font-size: 1rem/50px(对应UI屏宽750) 

### 关于引用
有使用到有赞[Vant](https://github.com/youzan/vant)的`Switch`、`Swipe`组件
